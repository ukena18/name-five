# Name Five

A mobile-first quick-thinking game with 150 prompt cards, built with React and Vite. It is a Progressive Web App (PWA), so players can install it directly from Chrome without an app store.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Netlify

Connect this folder to a Netlify site. The included `netlify.toml` configures the build automatically:

- Build command: `npm run build`
- Publish directory: `dist`

You can also run `npm run build` and drag the `dist` folder into Netlify Drop.

## Install on a phone

Open the deployed HTTPS site in Chrome, open the browser menu, and choose **Install app** or **Add to Home screen**.
